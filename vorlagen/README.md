# Vorlagen

Dieses Verzeichnis muss die Vorlagendateien (.docx / .xlsx / .pptx) enthalten.

## Erforderliche Dateien

Kopiere folgende Dateien aus dem WBI-Docu-Assist (`wbi-doku/vorlagen/`) in diesen Ordner:

| Dateiname | Verwendete Vorlage |
|-----------|--------------------|
| `Internes_Dokument_Vorlage.docx` | Internes Dokument (Word) |
| `Externes_Dokument_Vorlage.docx` | Externes Dokument (Word) |
| `Kundenanleitung_Vorlage.docx` | Kundenanleitung (Word) |
| `Internes_Dokument_Vorlage.xlsx` | Internes Dokument (Excel) |
| `Externes_Dokument_Vorlage.xlsx` | Externes Dokument (Excel) |
| `Netzwerkdoku_Vorlage.xlsx` | Netzwerkdokumentation (Excel) |
| `Präsentationsvorlage_Vorlage.pptx` | Präsentation (PPTX) |

## Hinweis

Ohne die Vorlagendateien kann kein Dokument exportiert werden.
Die App zeigt eine Fehlermeldung wenn die benötigte Vorlage fehlt.
